# 美国农业贸易分析（1990-2024）
本项目基于美国农业部（USDA）官方数据，对1990-2024年美国农业进出口贸易进行全面分析，涵盖贸易趋势、核心伙伴及市场格局等关键维度，形成自动化数据处理与可视化分析闭环。

## 项目概述
本项目聚焦美国农业进出口领域，以USDA发布的“年度Top15进口来源国”和“年度Top15出口目的地”数据为核心，通过数据清洗、统计分析与可视化，揭示美国农业贸易的长期规律与年度特征，为理解美国农业国际竞争力提供数据支撑。

### 核心目标
1. 实现全流程自动化（文件夹创建、数据下载、清洗与分析）
2. 将USDA宽格式数据转换为长格式，适配分析需求
3. 挖掘1990-2024年农业进出口长期趋势与波动特征
4. 识别各年度Top15贸易伙伴，分析市场集中度
5. 计算贸易差额，评估美国农业贸易平衡状态

## 数据来源
- **提供机构**：美国农业部（USDA）经济研究局（ERS）
- **数据集名称**：美国对外农业贸易统计（FATUS）
- **时间范围**：1990-2024财年
- **官方下载地址**：
  - 进口数据：[Mfytop15.xlsx](https://ers.usda.gov/sites/default/files/_laserfiche/DataFiles/50441/Mfytop15.xlsx?v=26152)
  - 出口数据：[xfytop15.xlsx](https://ers.usda.gov/sites/default/files/_laserfiche/DataFiles/50441/xfytop15.xlsx?v=60078)
- **数据维度**：各年度Top15贸易伙伴的贸易金额（以美元计）


## 数据处理流程
### 1. 环境初始化
运行 `create_project_structure.py` 脚本，自动生成项目所需文件夹与空 Notebook 文件：
```bash
python create_project_structure.py
```

### 2. 数据下载与清洗（`01_data_clean.ipynb`）
- **自动下载**：通过代码调用USDA官方链接，将原始Excel文件保存至 `data_raw/`；若网络失败，支持手动下载后放入该文件夹。
- **格式修复**：跳过原始文件前2行无效表头，删除空列，确保数据结构规范。
- **宽转长处理**：将“年份为列”的宽格式数据，转换为“年份为行”的长格式（列：`Country`（国家）、`fiscal_year`（财年）、`value_usd`（金额）、`trade_type`（贸易类型））。
- **数据校验**：过滤空国家、金额为0或负数的无效数据，保留各年度Top15贸易伙伴记录。

### 3. 统计分析（`02_data_analysis.ipynb`）
- **年度汇总**：计算各年度进口总额、出口总额及贸易差额（出口-进口），生成年度趋势表。
- **Top15分析**：提取2024年（或最新可用年份）Top15进口/出口伙伴，计算其市场占比。
- **核心伙伴追踪**：聚焦墨西哥、加拿大、中国、欧盟（EU-27）四大核心伙伴，分析其与美国的贸易趋势。
- **可视化生成**：自动输出趋势图、柱状图等图表，保存至 `output/charts/`。

### 4. 最终报告（`00_analysis_report.ipynb`）
- 整合所有分析结果，形成结构化报告，包含：
  - 贸易趋势总结（长期与年度）
  - Top15伙伴排名及市场集中度分析
  - 贸易平衡状态评估
  - 嵌入式可视化图表
  - 研究局限与后续建议


## 核心分析结论（基于2024年USDA官方数据）
### 1. 整体贸易格局
- **2024年进口总额**：2062.07亿美元（主要品类：新鲜农产品、畜牧产品、热带水果），较2023年（1953.34亿美元）增长5.56%。
- **2024年出口总额**：1744.28亿美元（主要品类：谷物、大豆、乳制品、肉类），较2023年（1781.84亿美元）小幅下降2.11%。
- **2024年贸易差额**：-317.79亿美元（贸易逆差），较2023年逆差（171.51亿美元）有所扩大，主要受出口小幅下滑、进口持续增长影响。
- **长期趋势**：1990-2024年进口额从238.16亿美元增长至2062.07亿美元，复合年增长率（CAGR）约7.8%；出口额从453.48亿美元增长至1744.28亿美元，复合年增长率约5.2%，进口增速长期高于出口。

### 2. Top15贸易伙伴
#### 2024年进口Top5来源国（含具体金额与占比）
1. **墨西哥**：478.72亿美元，占2024年总进口额的23.22%（主要商品：牛油果、番茄、猪肉、新鲜蔬菜）。
2. **加拿大**：405.78亿美元，占总进口额的19.68%（主要商品：牛肉、油菜籽、乳制品、小麦）。
3. **欧盟-27**：352.51亿美元，占总进口额的17.10%（主要商品：乳制品、葡萄酒、肉类加工品）。
4. **巴西**：72.75亿美元，占总进口额的3.53%（主要商品：咖啡、糖、牛肉）。
5. **中国**：56.47亿美元，占总进口额的2.74%（主要商品：海产品、新鲜水果、茶叶）。

#### 2024年出口Top5目的地（含具体金额与占比）
1. **墨西哥**：300.36亿美元，占2024年总出口额的17.22%（主要商品：玉米、大豆、猪肉、乳制品）。
2. **加拿大**：290.04亿美元，占总出口额的16.63%（主要商品：大豆、小麦、牛肉、饲料）。
3. **中国**：257.45亿美元，占总出口额的14.76%（主要商品：大豆、玉米、棉花、高粱）。
4. **欧盟-27**：123.90亿美元，占总出口额的7.10%（主要商品：大豆、小麦、肉类）。
5. **日本**：118.51亿美元，占总出口额的6.80%（主要商品：玉米、小麦、牛肉、乳制品）。

### 3. 市场集中度特征
- **进口集中度**：Top3来源国（墨西哥、加拿大、欧盟-27）合计占总进口额的60.00%，Top5来源国合计占比69.27%，对北美及欧盟供应链依赖度极高。
- **出口集中度**：Top3目的地（墨西哥、加拿大、中国）合计占总出口额的48.61%，Top5目的地合计占比62.49%，区域市场集中特征显著，中国作为单一最大非北美出口市场，对美国农业出口影响关键。

### 4. 核心伙伴贸易趋势（1990-2024）
- **墨西哥**：进口额从1990年26.64亿美元增长至2024年478.72亿美元（CAGR 9.2%），出口额从1990年27.01亿美元增长至2024年300.36亿美元（CAGR 8.3%），受NAFTA/USMCA协定推动，长期保持双位数增长，是美国农业第一大贸易伙伴。
- **加拿大**：进口额从1990年3.63亿美元增长至2024年405.78亿美元（CAGR 11.5%），出口额从1990年37.72亿美元增长至2024年290.04亿美元（CAGR 6.7%），进口增速远超出口，成为美国农业进口第一大来源国。
- **中国**：进口额从1990年无显著数据（不足1亿美元）增长至2024年56.47亿美元，出口额从1990年不足5亿美元增长至2024年257.45亿美元，2010-2020年期间波动剧烈（受贸易政策影响），2024年出口额较2023年（336.43亿美元）下降23.48%。
- **欧盟-27**：进口额从1990年5.29亿美元增长至2024年352.51亿美元（CAGR 10.1%），出口额从1990年8.45亿美元增长至2024年123.90亿美元（CAGR 7.5%），进口增长持续高于出口，贸易逆差逐年扩大。

## 研究局限与后续方向
### 研究局限
1. 数据仅包含Top15贸易伙伴，未覆盖中小规模贸易关系，可能遗漏区域细分市场特征。
2. 聚焦贸易金额（名义值），未纳入贸易量、产品单价等维度，也未剔除通胀影响。
3. 未结合政策事件（如关税调整、疫情）进行因果分析，仅呈现趋势相关性。

### 后续研究方向
1. 加入通胀调整（使用CPI将名义金额转换为实际金额），更准确反映贸易量变化。
2. 细分产品维度分析（如大豆、玉米、牛油果等单一品类的进出口格局）。
3. 结合政策事件构建计量模型，量化贸易政策对进出口的影响程度。
4. 使用时间序列模型（如ARIMA、Prophet）预测未来3-5年美国农业贸易趋势。
